# Skill2Cash

Welcome to the Skill2Cash platform! This project aims to connect skill providers with individuals seeking to learn or hire those skills. Through our platform, users can offer their expertise, create a portfolio, and earn money by teaching or consulting.

## Project Overview

The Skill2Cash platform is designed to facilitate skill sharing and monetization. Users can create profiles, list their skills, set rates, and connect with others who are interested in learning or hiring those skills.

### Key Features:
- User registration and profile management
- Skill listing with descriptions and pricing
- Search functionality to find available skills
- Reviews and ratings for skill providers
- Secure payment processing
- Messaging system for communication

## Deployment Instructions

To deploy Skill2Cash, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/oedwin/skill2sente.git
   cd skill2sente
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up the environment variables by creating a `.env` file based on the provided `.env.example`.
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and access `http://localhost:3000`.
